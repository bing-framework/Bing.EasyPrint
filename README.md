# Bing.EasyPrint
[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://mit-license.org/)

Bing.EasyPrint是Bing应用框架的便携式打印机命令生成库。
主要用于热敏、小票打印相关功能操作。已支持CPCL指令集的生成。
